# Portfolio Website - Rachit Prajapati

Welcome to the source code for my professional portfolio. This website is designed to showcase my journey as a **Backend Engineer** and my passion for building scalable, high-performance systems.

## 🚀 About the Project
This portfolio is built to reflect my technical philosophy: **clean architecture, high integrity, and user-centric design**. It serves as a central hub for my professional experience, technical skills, and featured projects.

## 🛠 Tech Stack
The frontend of this portfolio utilizes a modern, performance-first stack:
* **Framework:** Tailwind CSS for rapid, responsive UI development.
* **Animations:** * `Particles.js` for dynamic, interactive backgrounds.
    * `Typed.js` for professional typing effects.
* **Icons:** `Devicon` for high-quality developer tool branding.
* **Deployment:** Hosted via GitHub Pages.
